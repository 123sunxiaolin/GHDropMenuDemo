# GHDropMenuDemo
电商筛选菜单

超级简单好用的电商筛选菜单 

![Untitled.gif](https://upload-images.jianshu.io/upload_images/1419035-0ce48e808c865eb4.gif?imageMogr2/auto-orient/strip)

* 自定义标题,自定义标题内容
* 动画展开,移除
* 可以重新传入模型,刷新数据源
* 选中内容通过代理的方式回调
* 对原有项目无污染,直接拖进项目即可使用
